Seance 1.0

A Medium-inspired theme for Ghost.

Check out maxjackson.online for a very good example of the Seance theme in action :D.

Features

  • Mobile-first design
  • Fully responsive layout
  • Compatible with Ghost 1.14.1
  • Article header images
  • Automatic figcaption generation from image alt tags
  • Extremely valid HTML5/CSS3
  • WAI-ARIA & Rich Snippets(microdata) roles
  • Static page support
  • Featured article support
  • Google Analytics integration
  • FontAwesome
  • Highlight.js integration for Syntax highlighting/li>
  • OpenGraph & Twitter Cards meta’s
  • Free & open source font usage

Installation

  1. Upload your .zip file via the 'Upload a theme' button at the bottom of the Ghost design panel (yourdomain.com/ghost/#/settings/design).
  2. Switch your theme to the Seance theme.
  3. Have fun!
